# Como usar os templates no Confluence

Os arquivos nesta pasta estão em **Confluence Wiki Markup**. Use um dos métodos abaixo para colar no Confluence.

---

## Método 1: Inserir como Markup (recomendado)

1. No Confluence, crie uma **nova página** ou **edite** uma existente.
2. No editor, abra o menu **"..."** (três pontos) ou **Insert**.
3. Escolha **"Insert markup"** ou **"Markup"**.
4. Selecione **"Confluence Wiki"** (ou "Wiki Markup").
5. Abra o arquivo `.confluence.txt` desejado, copie **todo** o conteúdo (Ctrl+A, Ctrl+C).
6. Cole na caixa de markup e clique em **"Insert"** ou **"Apply"**.

As tabelas, listas, negrito e checkboxes serão formatados corretamente.

---

## Método 2: Colar direto (Confluence Cloud)

Em **Confluence Cloud**, às vezes é possível:
1. Copiar o conteúdo do `.confluence.txt`.
2. Colar diretamente no corpo da página.

Se a formatação não ficar boa, use o **Método 1**.

## Dica

Para criar uma **página de template** no Confluence:
1. Crie uma página com o conteúdo do template (usando o Método 1).
2. Salve.
3. Use **"Create from template"** nas próximas vezes ou copie o conteúdo dessa página para novas páginas.
