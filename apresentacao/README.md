# Apresentação - Processo de Qualidade QA

## Como Usar

### Abrir a Apresentação

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. A apresentação abrirá em tela cheia automaticamente

### Navegação

**Teclado:**
- **→** ou **Espaço**: Próximo slide
- **←**: Slide anterior
- **Home**: Primeiro slide
- **End**: Último slide

**Mouse:**
- Botões de navegação na parte inferior
- Clique em "Próximo →" ou "← Anterior"

**Touch (tablets/mobile):**
- Deslize para a esquerda: Próximo slide
- Deslize para a direita: Slide anterior

### Características

- **38 slides completos** cobrindo todo o processo
- **Design moderno e profissional**
- **Indicador de progresso** no topo
- **Navegação intuitiva**
- **Responsivo** (funciona em diferentes tamanhos de tela)
- **Animações suaves** entre slides

### Dicas para Apresentação

1. **Teste antes**: Abra a apresentação antes da reunião para garantir que funciona
2. **Modo tela cheia**: Use F11 para tela cheia no navegador
3. **Backup**: Tenha os documentos originais como backup
4. **Tempo estimado**: Apresentação completa leva aproximadamente 30-45 minutos
5. **Q&A**: Reserve tempo para perguntas ao final

### Personalização

Para personalizar a apresentação:
- Edite o arquivo `index.html`
- Modifique cores, textos ou adicione slides conforme necessário
- As cores principais estão definidas em `#667eea` (azul) e `#764ba2` (roxo)
