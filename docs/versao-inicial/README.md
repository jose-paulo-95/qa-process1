# Versão Inicial - Processo Simplificado

## Visão Geral

Esta é a **versão inicial** do processo de QA, pensada para equipes que querem começar com menos burocracia e um fluxo mais ágil.

**Inspirado no feedback do coordenador**: menos burocracia na entrada de demandas e Kanban com menos colunas.

---

## Quando usar esta versão

- Equipes que estão começando o processo de QA
- Quando se busca agilidade em vez de formalidade
- Para times menores ou com comunicação muito próxima
- Quando o coordenador pede praticidade

---

## O que esta versão oferece

| Aspecto | Versão Inicial | Versão Final |
|---------|----------------|--------------|
| **Colunas Kanban** | 3 colunas | 6 colunas |
| **Entrada de demandas** | Formulário essencial | Template completo |
| **Cerimônias** | Apenas Daily | Planning, Daily, Retro |
| **Métricas** | Básicas | Completas com KPIs |
| **Documentação** | Mínima necessária | Completa e padronizada |

Quando a equipe amadurecer e sentir necessidade de mais controle e visibilidade, migre para a **versão final** em `docs/` (metodologia-trabalho.md e processo-qualidade.md).
