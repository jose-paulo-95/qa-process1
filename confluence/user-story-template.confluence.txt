h2. Definition of Ready (DoR)

A story está pronta para ser trabalhada quando:

* ( ) Requisitos claros e documentados
* ( ) Critérios de aceite definidos
* ( ) Story estimada pela equipe
* ( ) Dependências identificadas e resolvidas
* ( ) Design/UX aprovado (quando aplicável)
* ( ) Ambiente de desenvolvimento disponível
* ( ) Acesso necessário concedido
* ( ) Dados de teste definidos
* ( ) Priorizada pelo Product Owner
* ( ) Aprovada para entrar no fluxo

----
