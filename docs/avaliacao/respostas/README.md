# Respostas de Avaliação - QA

Esta pasta armazena as respostas das avaliações de competências (autoavaliação e avaliação do gestor).

## Arquivos

- **resp-junior-autoavaliacao.xlsx** - Respostas da autoavaliação QA Junior (baseline para PDI)

## Uso

As respostas são utilizadas como baseline para:
- Identificação de gaps de competências
- Elaboração do Plano de Desenvolvimento Individual (PDI)
- Acompanhamento de evolução ao longo do tempo

## Reavaliação

Recomenda-se preencher novamente a autoavaliação a cada 6-12 meses para comparar o progresso com o baseline inicial.
