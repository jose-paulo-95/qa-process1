# Plano de Desenvolvimento Individual (PDI) - QA Junior

Esta pasta contém os materiais de apoio para execução do PDI baseado na autoavaliação.

## Estrutura

- **perfil-contexto-pdi.md** - Resumo do que foi declarado na autoavaliação (para mentor/gestor)
- **template-mentoria.md** - Template para agendamento e acompanhamento de sessões de mentoria
- **fase1-checklist.md** - Checklist e recursos da Fase 1 (Fundamentos + Postman)
- **boas-praticas-qa.md** - Boas práticas, técnicas de QA e dicas para "ser mais prático"
- **glossario-qa.md** - Glossário de termos técnicos para comunicação

## Projeto de Prática

A pasta [postman-serverest/](postman-serverest/) contém:
- Collection Postman para a API ServeRest
- Environment com variáveis
- README com cenários de teste sugeridos

Utilizado como exercício prático na Fase 1 do PDI.

## Curso Completo

O [curso-pleno/](curso-pleno/) contém 6 módulos com explicações, exercícios e provas para evolução Junior → Pleno.

**Aplicação Web:** O curso também está disponível como aplicação Next.js em [curso-web/](../../../curso-web/). Execute `npm run dev` na pasta curso-web para acessar em http://localhost:3000
