# Swagger: leitura e documentação

---

## OpenAPI/Swagger

Especificação para descrever APIs REST. Permite ver endpoints, métodos, parâmetros, body, respostas.

---

## O que identificar

- Base URL
- Endpoints e métodos
- Parâmetros (query, path, body)
- Schemas (campos, tipos, obrigatórios)
- Códigos de resposta (200, 400, 401, 404, 500)
