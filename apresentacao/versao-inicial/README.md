# Apresentação - Processo de Qualidade QA (Versão Inicial)

## Como Usar

### Abrir a Apresentação

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Use F11 para tela cheia, se desejar

### Navegação

**Teclado:**
- **→** ou **Espaço**: Próximo slide
- **←**: Slide anterior
- **Home**: Primeiro slide
- **End**: Último slide

**Mouse:**
- Botões "Próximo →" e "← Anterior" na parte inferior

**Touch (tablets/mobile):**
- Deslize para a esquerda: Próximo slide
- Deslize para a direita: Slide anterior

### Conteúdo da Apresentação

1. **Capa e Apresentação** (Slides 1-2)
2. **Quando usar** (Slide 3)
3. **Entrada de Demandas** (Slides 4-8): princípio, 3 campos, exemplo, priorização
4. **Kanban** (Slides 9-10): mesmo modelo (Backlog → … → Done) e fluxo
5. **Cerimônias e Papéis** (Slides 11-12)
6. **Definição de Pronto e Evolução** (Slides 13-15)
7. **Resumo e Perguntas** (Slides 16-17)

### Tempo estimado

Aproximadamente **15–20 minutos** para apresentar a versão inicial.
